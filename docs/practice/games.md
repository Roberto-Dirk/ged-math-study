# 🎮 Educational Games

Learning math is more fun with games! Challenge yourself and compete for high scores.

## Available Games

### 🏃 Math Speed Challenge
Race against the clock to solve basic math problems. Build speed and accuracy!

- **Difficulty levels**: Easy, Medium, Hard
- **Time limits**: 60s, 120s, 180s
- **Leaderboard**: See how you rank

### 🧮 Equation Master
Solve equations step-by-step. Get hints or go solo!

- **Topics**: Linear, quadratic, systems
- **Modes**: Guided, Challenge
- **Rewards**: Badges for streaks

### 🔷 Geometry Puzzle
Visualize and solve geometric problems.

- **Puzzles**: 50+ unique challenges
- **Hints**: Available for tricky ones
- **Difficulty progression**: Unlocks as you advance

### 📊 Data Detective
Analyze data and find patterns. Multiple choice and free response!

- **Datasets**: Real-world scenarios
- **Analysis tools**: Built-in calculator
- **Tips**: Learn as you play

### 🎲 Probability Planner
Understand probability through interactive simulations.

- **Simulations**: Dice, coins, spinners
- **Experiments**: Run your own probability tests
- **Theory vs Reality**: Compare predictions to results
